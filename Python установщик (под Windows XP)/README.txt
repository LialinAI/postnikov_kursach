1. Распаковать архив
2. Установить C++ (если не был установлен)
3. Установить Python
4. Если есть интернет, установить pysimplegui
pip install pysimplegui
4. Если нет интернета, то
pip install PySimpleGUI-4.60.5-py3-none-any.whl
ИЛИ скопировать off-line  версии в каталоги с программами (каталог pysimplegui)
5. запускать py файлы как обычный exe
